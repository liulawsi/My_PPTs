存放你的程式碼。(work space)
也可以是一個紙本筆記本

或者兩個都準備
讓我們開始入門的第一課
首先先認識電腦
電腦概略的劃分可以分為五大單元
控制單元、算術與邏輯單元、
輸入單元、輸出單元以及記憶單元
初學的你不用知道太多專業知識
初學程式的你只要關心三件事
輸入、運算與輸出
首先
輸入就是人機介面輸入
或者是從儲存裝置或網路輸入
輸入的內容可能是數字、文字或是訊號等等

再來
運算則是包括基的[加 減 乘 除]這類運算
以及像是[比較大小]的邏輯預算
還有發展出解決問題的複雜演算法

最後
輸出則是把資料或文字輸出到人機介面
如螢幕、印表機、喇叭等
又或者輸出到儲存裝置或網路

白羊老師認為 學程式
最好的順序就是從輸出開始上手

在前面的單元介紹開發環境時
當時用來測試的第一個程式
顯示了 Hello Python
這就是一個最簡單的輸出範例
print就是一個叫電腦輸出的指令
指令後面接著一對括號
括號內放入要讓指令執行的資料
也就是參數
指令依照你給他的參數做出相對應的事
你給他文字 Hello Python
那麼print指令就會把Hello Python 顯示出來
如果你把Hello Python改為This is a book.
那麼他就會顯示This is a book,

補充一點
想要它顯示的文字、數字、符號或空白
Python規定必須用兩個雙引號包在中間
括號與雙引號都是成對出現的
千萬不要漏掉一邊喔


簡單明瞭吧！

好吧 解答如下
print沒辦法讓你的輸出直的顯示！

他穆罕默德宣稱
他可以叫山移至他的面前來

既然print只能橫著顯示內容
那我就換個方式去適應它

再詳細的說一下
print所提供的螢幕的輸出功能
他其實就像個搬運工
他會把你給他的文字
一個一個搬到一個你看不到的格子上
這種格子的輸出一般稱為終端機
print把自搬到位置放好後
他會換到下一行等著下一次輸出
而且他做過就不回頭了
也就是當他開始在第二行等待輸出時
第一行就不能再增加或改變了

掌握好這個特性
相信你就可以好好的使用這種輸出方式了

這個指令還有很多其他強大功能
先不要急著學
他只會讓你頭很大很痛
後面我們會在需要時一一說明
